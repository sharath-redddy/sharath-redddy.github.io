# Sharath Reddy — Data Analyst Portfolio

A polished, responsive single-page portfolio .
## Included
- Premium responsive portfolio (`index.html`, `styles.css`, `script.js`)
- Embedded project and certification links
- Resume PDF in `assets/`

## Local preview
Open `index.html` in a browser, or run:
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000`.
