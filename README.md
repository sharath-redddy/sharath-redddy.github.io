# Sharath Reddy — Data Analyst Portfolio

A polished, responsive single-page portfolio showcasing SaaS churn analytics, demand forecasting, inventory optimization, and customer segmentation work.

## Included
- Premium responsive portfolio (`index.html`, `styles.css`, `script.js`)
- Embedded project and certification links
- Resume PDF in `assets/`
- No framework or build process required
- Optimized for GitHub Pages

## Publish with GitHub Pages
1. Create a new **public** repository named exactly: `sharath-redddy.github.io`
2. Upload the **contents inside this folder** — do not upload the folder itself as a nested folder.
3. On GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main** and folder **/(root)**, then click **Save**.
6. Wait a few minutes. Your site will be available at: `https://sharath-redddy.github.io`

## Customize later
- Update text and links in `index.html`
- Adjust visual styles in `styles.css`
- Replace the file in `assets/Sharath_Reddy_Data_Analyst_Resume.pdf` when you update your resume

## Local preview
Open `index.html` in a browser, or run:
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000`.
