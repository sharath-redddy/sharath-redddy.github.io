## How to update your current GitHub Pages portfolio

1. Download and unzip the package.
2. Open your portfolio repository: `sharath-redddy.github.io`.
3. Delete or replace the old `index.html`, `styles.css`, `script.js`, and `assets` files.
4. Upload all files from this folder to the root of the repository.
5. Commit the changes.
6. Wait a few minutes, then refresh: `https://sharath-redddy.github.io/`.
